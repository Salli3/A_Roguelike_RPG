# A "Rock"like RPG

Just a simple topdown rpg game

## About

A small 2D room-based shooter built in Unity. Fight through increasingly difficult enemy room, choose upgrades to strengthen your character, and survive as long as possible.

## Features

- Multiple enemy types with different behaviours
- Difficulty-scaling budget-based enemy spawner​
- Upgrade system

## Controls 

WASD - Move 

Left Mouse - Attack / Interact 

Left Shift - Dash

ESC - Pause 

## Built With

- Unity
- C#
